# Inventory Management API

A simple REST API built with FastAPI for managing inventory items with basic CRUD operations.

## Tech Stack

* FastAPI
* Python
* SQLite

## Features

* Create, read, update, and delete items
* Structured project using routers, models, and services
* Built-in API docs with Swagger UI

## Setup

```bash
git clone https://github.com/Himanshu-gautam738/Inventory-management-FastAPI.git
cd Inventory-management-FastAPI
pip install -r requirements.txt
uvicorn app.main:app --reload
```
